Stellarjob

You're probably seeing this page because someone +++'d you via the Stellarjob bot. This means that the sender thinks you did something great, and wants to tell the world about it.

To accept your +++, you need to do the following:

  1. Create a Stellar account at https://launch.stellar.org, if you haven't already.
  2. Trust the Stellarjob bot to issue you [amount] +++. You can do this by entering your username and password in the box below (the credentials are never sent outside of your browser), or by making a curl request as shown below.

$ curl -X POST https://test.stellar.org:9002  -d '
{
    "method": "submit",
    "params": [
        {
            "secret": "<USER_SECRET>",
            "tx_json": {
                "Account": "<USER_ACCOUNT>",
                "LimitAmount": {
                    "currency": "USD",
                    "issuer": "<GATEWAY_ACCOUNT>",
                    "value": 30
                },
                "TransactionType": "TrustSet"
            }
        }
    ]
}
'
